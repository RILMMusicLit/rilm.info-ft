Documentation für das Autoren-Dashboard

Das Editorial System ist web-basiert und erfordert lediglich einen Web Browser und Internet-Zugang. Das System unterstützt aktuelle Versionen von Firefox und Google Chrome. Wir empfehlen jedoch dringend, das Tool via Chrome zu öffnen und zu nutzen. Dazu müssen Sie sich eine aktuelle (kostenfreie) Version von Google Chrome auf den Rechner laden.

Allgemeines

Das Dashboard gibt Autoren, Übersetzern und anderen Mitarbeitern Zugang zu den Artikeln, mit deren Revision/Neufassung sie beauftragt wurden.

Der Begriff »Artikel« meint sowohl komplette Artikel als auch Teile eines Artikels (der von verschiedenen Autoren verfasst und daher in die entsprechenden Teile aufgeteilt wurde).

Auf der Publishing Platform erscheinen die Teile eines Artikels nicht als Teile sondern innerhlab des gesamten Artikels.

Es gibt sechs Menüs zu den Artikeln, auf die über das Dashboard zugegriffen werden kann:

View all articles: Alle Artikel ansehen

Articles with recent messages: Artikel mit neuen Nachrichten

Open articles: Offene Artikel

Assigned articles: Zugeteilte Artikel

Collaboration: Zusammenarbeit

Past assignments: Vergangene Aufträge

Bei einem Klick auf den entsprechenden Button oben links auf dem Bildschirm erscheint die entsprechende Liste. Öffnet man das Dashboard, erscheint zuerst das Menü Open articles.

List of Articles

Wurden Artikel geteilt, zeigt das Feld Section genau an, mit welchem Teil des Artikels Sie beauftragt wurden. Bei nicht geteilten Artikeln zeigt dieses Feld immer die Angabe “Part 0”.

Es gibt zwei Arten von Mitteilungen, welche in dem System gespeichert werden: Öffentliche Nachrichten (und Anmerkungen) sind für alle diejenigen sichtbar, die mit der Bearbeitung des Artikels beauftragt wurden, sowie für die MGG-Redakteure. Die Anzahl dieser Nachrichten (seit der letzten Veröffentlichung oder Übersetzung) erscheint im Dashboard in der General Spalte. Private Nachrichten können nur von dem Sender und dem Empfänger eingesehen werden. Die Private Spalte zeigt die Anzahl der privaten Nachrichten, welche Sie gesendet oder empfangen haben.

Klicken Sie auf eine Zeile, um mehr Informationen zu einem Artikel zu sehen oder um ihn bearbeiten zu können (wenn Sie Zugang haben). Rechts finden Sie Buttons, mit denen Sie verschiedene Aktionen abschließen können sowie zwei Felder Details und Document History mit weiteren Informationen über den Artikel.

Buttons

  • Wenn Sie einen Artikel in dem Dashboard auswählen, erscheinen verschiedene Buttons unter dem Titel des Artikels auf der rechten Seite des screens:
  • Submit gibt den Zugang zu dem Artikel an die Redaktion zurück.
  • Im Falle, dass mehr als ein Nutzer (Autor oder Übersetzer) an einem Teil eines Artikels zusammen arbeitet, erlaubt Ihnen der Button Collaborate einem anderen Nutzer Zugang zu gewähren (mit derselben Funktion). Der Zugang zu dem Artikel kann damit hin und her geschickt werden.
  • Edit öffnet den Artikel im Editor, d.h. in der Bearbeitunsgfunktion.
  • View öffnet den Artikel lediglich zur Ansicht oder zum Druck.
  • Add Note erlaubt Ihnen, eine Anmerkung oder Nachricht zu dem ausgewählten Artikel zu schreiben. Einen Nachricht ist an einen bestimmten User oder Editor im System adressiert. Eine Anmerkung ist nicht an eine spezielle Person adressiert. Wenn eine Nachricht als Private markiert ist, kann sie nur der Verfasser und der Empfänger sehen.
  • Approve erlaubt einem User (Autor oder Editor), einen Artikel zur Veröffentlichung freizugeben.

Sie können nur diejenigen Artikel bearbeiten, zu denen Sie Zugang bekommen haben. Es kann immer nur ein User zu einer bestimmten Zeit Zugang zu einem Artikel haben, nicht zwei oder mehrere gleichzeitig. Sie können jeden Artikel sehen und Anmerkungen schreiben, zu dem Sie Zugang erhalten haben.

Document History

Um Anmerkungen und Nachrichten zu lesen, die zu einem bestimmten Artikel verfasst wurden, klicken Sie auf See more am Ende der Document History section.
