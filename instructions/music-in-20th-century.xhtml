Encyclopedia of Music in the 20th Century

rev'd 6/30

  1. General comments:
    • This publication should be converted from the provided XML and PDF.
    • American-style quotation marks are used throughout and are encoded as &#x201C; and &#x201D; in the source. Use "American rule" for single quotation marks.
    • Convert all other "&#" codes to Unicode characters. No "&#" codes should remain in the TEI files.
    • All <xref ref-type="page"> elements should be converted to the TEI element <pb>:
      <xref ref-type="page" id="page_443" />

      <pb n="443"/>
    • Convert all <italic> tags to <title>.
    • Ignore all <alt-title> elements and their contents.
    • Except for the front matter, each <book-part> element in the source will correspond to one TEI file;
    • Each <sec> element in the source will correspond to a <div> in TEI, but they might be nested differently.
    • See the table below for the value of @type in <text> for the listed entries. All other entries (not on the list) will be <text type="person">. All will have <div type="heading">, <div type="body">, <div type="authors">, and <div type="ography">.
    • Many articles will have a <div type="ref"> between the "authors" and "ography".
  2. Front matter will include the title page, copyright page, and Introduction.
    • The title page will correspond with the source elements <book-title-group> and <contrib-group>.
    • The copyright page will correspond with the source <publisher>, <isbn>, <pub-date>, <permissions>, and <imprint-meta> elements.
    • The Introduction is tagged as <book-part book-part-type="chapter" id="chapter1">.
  3. Entries correspond with <book-part> elements with @book-part-type="chapter".
    • For the <div type="heading">, use the contents of the source <title>.
    • In person entries, tag the name as <rilmPerson> and <persName> as usual. The part of the name after the last space can be tagged <surname> and the others preceding it as <forename>.
      <title>Hans Werner Henze</title>
      <div type="heading"><p><rilmPerson xml:id="r1"><persName><forename>Hans</forename> <forename>Werner</forename> <surname>Henze</surname></persName></rilmPerson></p></div>
    • Entries begin with a Drop Cap in print, which is tagged <bold> in the source XML. Omit the <bold> tags around the first letter of the first <p>.
  4. In-line Cross-references are indicated only by <sc> tags in the source XML. Set these references as a <ref> element, targeting the approriate article. Convert the contents of the <sc> tags to lower case and omit the <sc> tags.
    <p>Henze wrote his first symphony in 1947, the piece showing the influence of the neoclassicists Paul H<sc>INDEMITH</sc> and Igor S<sc>TRAVINSKY</sc>. ..</p>
    <p>Henze wrote his first symphony in 1947, the piece showing the influence of the neoclassicists Paul <ref target="#emt0000?">Hindemith</ref> and Igor <ref target="#emt0000?">Stravinsky</ref>. ..</p>
  5. The author name is given in a <sig-block> element in the source XML. Convert to a <div type="authors">.
  6. See-also references in the source follow the author name. They are tagged in a <p> element that begins with
    <p>S<sc>EE ALSO</sc>:
    Each reference is in <xref> tags in the source. Convert these to <ref> tags in TEI
  7. Most entries will have two <sec> elements with <title> values "Further Reading" and "Suggested Listening". Convert these to <div type="ography"> elements. Convert these according to the Taylor & Francis crosswalk.
    • Convert source <ref-list> elements to <list>.
    • Convert <ref> elements to TEI <item> and remove the attributes.
