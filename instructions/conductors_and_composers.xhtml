  1. General comments:
    • This publication should be converted from the provided XML and PDF.
    • English-style single quotation marks (‘ and ’) are used throughout and are encoded as &#x2018; and &#x2019; respectively in the source. The latter code is used for both right quote and for apostrophe, but they have to be distinguished in TEI:
      master&#x2019;s degree ... &#x2018;Jazz Pizzicato&#x2019;,
      master’s degree ... <q>Jazz Pizzicato</q>,
      Convert all other "&#" codes to Unicode characters. No "&#" codes should remain in the TEI files.

    • Cross-references are indicated by <sc> tags in the source XML. Set these references as a <ref> element, targeting the approriate article. Omit the <sc> tags.

  2. Front matter will include the title page, copyright page, dedication, Authors' Note, Keys to Abbreviation, and Acknowledgments.
    • The title page will correspond with the source elements <book-title-group> and <contrib-group>.
    • The copyright page will correspond with the source <publisher>, <isbn>, <pub-date>, <permissions>, and <imprint-meta> elements.
    • The dedication page content will correspond with the <notes notes-type="dedication"> of the source.
    • The Authors' Note is tagged <sec sec-type="fm-chapter" id="Authors-Note"> in the source. Use <title> as <head> and set the <sig-block> as a <docAuthor>.
  3. Article titles give the name in surname, forename order.
    • For the <div type="heading">, tag the name as <rilmPerson> and <persName>. The part of the name after the last space can be tagged <surname> and the others preceding it as <forename>.
      <div type="heading"><rilmPerson xml:id="r1"><persName><forename>June</forename> <surname>Allyson</surname></persName></rilmPerson></div>
  4. There may be any number of <div type="ography"> elements. They will correspond with <sec> elements in the source.
    • Keep source <list> elements as <list> but remove the attributes.
    • <list-item> elements will become TEI <item><bibl> elements.
    • Information can be tagged with <title>, <publisher>, and <idno type="label">.
      <item><bibl><title>International Showtime</title> (<publisher>NBC</publisher>, <date>1961—65</date>)</bibl></item>
      <item><bibl><title>The Bickersons Rematch</title> (<publisher>Col</publisher> <idno type="label">G-30523</idno>) w. Frances Langford</bibl></item>
