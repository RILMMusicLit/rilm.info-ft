Broadway: Its History, People, and Places an Encyclopedia (bae)

  1. If the first paragraph contains birth and/or death information--signalled by "b." or "d." then the the <text> element should have @type="person" and the first paragraph should contain <rilmPerson> as a direct child:
    <text type="person" xml:lang="en" xml:id="bae0001">
    <body>
    <div type="body" xml:id="d1">
    <p><rilmPerson><hi rend="bold"><surname>YOUMANS</surname>, <forename>VINCENT</forename>.</hi> The quintessential 1920s composer; <birth>b. <placeName>New York, New York</placeName>, <date when="1898-09-27" when-custom="1898">September 27, 1898</date></birth>; <death>d. <placeName>Denver, Colorado</placeName>, <date when="1946-04-05">April 5, 1946</date></death>.</rilmPerson></p>
    All other articles should have <text type="topic">
  2. All articles should contain one <div>, which should have @type="body"
  3. The headwords of each article (the first words in each one) are in allcaps and bold. Keep them in all caps and tag them with <hi rend="bold">, as above.
  4. All cross-references are in small caps in the source files. Please convert them to Roman and tag them <ref>. The link them to the appropriate article with @target. (The contents of @targe will have to be entered after all documents have been created.)
  5. Apostrophes in the source XML file are all represented in the source file as U+0027;. Please convert these to U+2019. *Please note that right single-quotes are already encoded as U+2019--you should convert these to <q rend="single"> before you convert the apostrophes so it can all be done automatically.
  6. Block quotes: see above.
  7. See-references: see above.
  8. Titles with dates: There are many text strings that corform to the following pattern:
    Title (DD/MM/YY, # performances)
    For instance,
    Du Barry Was a Lady (12/6/39; 408 performances)
    These should all be tagged in the following manner:
    <bibl><title type="work">Du Barry Was a Lady</title> (<run quantity="408" unit="perf"><rilmPerformance type="first"><date when="1939-12-06" when-custom="1923">12/6/39</date></rilmPerformance>; 408 performances</run>)</bibl>
    Two-digit years are always 19--.
    *This tagging should be able to be done in an automated fashion.
  9. Index (lines 6148–16574): See above. Also, page numbers in boldface can be ignored.
  10. The following sections should each receive its own document: "ACKNOWLEDGMENTS" and "INTRODUCTION". In the Introduction, please tag all phrases in small caps as <ref> and link them to the appropriate document.
