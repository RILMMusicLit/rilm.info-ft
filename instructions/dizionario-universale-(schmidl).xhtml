Dizionario universale dei musicisti (Schmidl)
  1. General comments:
    • This publication should be converted from scanned print
    • Articles will have <div type="heading"> and <div type="body">
    • Double-angle quotation marks ( « » ) are used throughout.
    • Italic type is used to indicate titles, so tag with <title> rather than <hi>.
    • All entries are <text type="person">.
  2. Front matter:
    • make a file for the cover with only illustration;
    • make the TEI title "[cover]"
    • make a file for advertisements "[advertisements]"
    • title page
    • copyright page
    • Vol. 1 additionally has a "Prefazione" (Preface).
  3. Tag the entry heads as <div type="heading">, containing <rilmPerson> and <persName> elements. Most heads start with surnames but do not indicate where the forename starts!
    • In general, tag up to the first space character as <surname>, and the remaining words as <forename>:
      [Luciani Sebastiano Arturo.]
      <persName><surname>Luciani</surname> <forename>Sebastiano</forename> <forename>Arturo</forename></persName>.
      [Oddone-Sulli-Rao Elisabetta.]
      <persName><surname>Oddone-Sulli-Rao</surname> <forename>Elisabetta</forename><persName>.
    • IF the name begins with "Mac " (i.e. with a space after "Mac") or begins with "O’ ", use the first two words as <surname>:
      <persName><surname>Mac Dowel</surname> <forename>Edward</forename> <forename>Alexander</forename></persName>.
    • IF the name contains " y " (lower-case "y" with a space on both sides), use the first three words as surname:
      <persName><surname>Ocón y Rivas</surname> <forename>Edoardo</forename></persName>.
    • Parentheses in the head indicate variant name forms or titles. Tag each in its own <persName> element (but in the same <rilmPerson>). Don't tag words not capitalized. [Reimann (Reymannus, Reymanni) Matteo.]
      <rilmPerson><persName><surname>Reimann</surname></persName> (<persName><surname>Reymannus</surname></persName>,<persName><surname>Reymanni</surname></persName>) <persName><forename>Matteo</forename></persName></rilmPerson>
      [Molière (il cuí ver nome è Poquelin) Giovanni Battista.]
      <rilmPerson><persName><surname>Molière</surname></persName> (il cuí ver nome è <persName><surname>Poquelin</surname></persName>) <persName><forename>Giovanni</forename> <forename>Battista</forename></persName></rilmPerson>.
  4. See-references are indicated by "vedasi" or "vedi" as Surname Forename, vedasi Surname Forename.

The Supplemento volume contains both new entries and additions to entries already in vol. 1 or vol. 2. Additions to previous entries are marked with an asterisk preceding the heading: [*Abrahamsen Erico.]. New entries have no such asterisk.

  1. Treat new entries in the Supplement identically with entries in vols. 1 and 2 (with different TEI header, of course).
  2. For additions to previous entries, treat identically except in the <text> element, add a @corresp attribute pointing to the <text> @xml:id of the original entry:
    • <text type="person" xml:lang="it" xml:id="dum-0000x"> for the original entry
    • <text type="person" xml:lang="it" xml:id="dum-0000y" corresp="#dum0000x"> for the supplement entry.
