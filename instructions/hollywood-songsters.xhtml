Encyclopedia of American Gospel Music (eag)
  1. General comments:
    • This publication should be converted from the provided XML.
    • Except for the front matter, each <book-part> element in the source will correspond to one TEI file.
    • All entries are <text type="person">, and have <div type="heading">, <div type="subheading">, and usually several <div type="ography">.
    • Remove all <sc> tags.
    • All <xref ref-type="page"> elements should be converted to the TEI element <pb>:
      <xref ref-type="page" id="page_443" />

      <pb n="443"/>
    • Convert all <italic> tags to <hi rend="italic">.
    • Ignore all <alt-title> elements and their contents.
    • Ignore all <sec-meta> elements and their contents.
    • Use American rule for single quotation marks.
  2. Front matter will include the title page, copyright page, dedication, Authors' Note, Keys to Abbreviation, and Acknowledgments.
    • The title page will correspond with the source elements <book-title-group> and <contrib-group>.
    • The copyright page will correspond with the source <publisher>, <isbn>, <pub-date>, <permissions>, and <imprint-meta> elements.
    • The dedication page content will correspond with the <notes notes-type="dedication"> of the source.
    • The Authors' Note is tagged <sec sec-type="fm-chapter" id="Authors-Note"> in the source. Use <title> as <head> and set the <sig-block> as a <docAuthor>.
  3. Article titles give the name in forename-surname order.
    • For the <div type="heading">, tag the name as <rilmPerson> and <persName>. The part of the name after the last space can be tagged <surname> and the others preceding it as <forename>. The names "Cher" and "Sammy Davis Jr." are the only exceptions.
      <div type="heading"><rilmPerson xml:id="r1"><persName><forename>June</forename> <surname>Allyson</surname></persName></rilmPerson></div>
    • The <p> element immediately following the article title should be tagged <div type="subheading"> containing a <rilmPerson> with an @corresp attribute pointing to the xml:id of the <rilmPerson> element above it. Tag the birth and death dates, and omit the parentheses and "b." and "d." If another name version is given, it should be set in a <persName> element.
      <p>(b. Ella Geisman, South Bronx, New York, October 7, 1917)</p>
      <div type="subheading">
      <rilmPerson xml:id="r2" corresp="#r1"> <persName><forename>Ella</forename> <forename>Jennings</forename> <surname>Geisman</surname></persName>
      <birth><placeName>South Bronx, New York</placeName><date when="1917-10-07" when-custom="1917">October 7, 1917</date>, </birth></rilmPerson>
      </div>
  4. There may be any number of <div type="ography"> elements. They will correspond with <sec> elements in the source.
    • Keep source <list> elements as <list> but remove the attributes.
    • <list-item> elements will become TEI <item><bibl> elements.
    • Information can be tagged with <title>, <publisher> and <date> or <idno type="label">.
      <item><bibl><title>International Showtime</title> (<publisher>NBC</publisher>, <date>1961â€”65</date>)</bibl></item>
      <item><bibl><title>The Bickersons Rematch</title> (<publisher>Col</publisher> <idno type="label"G-30523</idno>) w. Frances Langford</bibl></item>

Please refer also to the provided xml samples.
