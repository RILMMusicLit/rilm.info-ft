  1. This publication should be converted from XML.
  2. All articles are <text type="person">
  3. Titles in prose text: All italicized phrases that begin with a capital letter followed by parentheses should be tagged together as <bibl>, the italized portion tagged as <title type="work"> and dates in the parentheses tagged in the following manner (all other italicized phrases should be tagged <hi rend="italic">):
    Source: <italic>A Woodland Serenade</italic> for wind quintet (1955)
    TEI-XML: <bibl><title type="work">A Woodland Serenade</title> for <instrumentation>wind quintet</instrumentation> (<date>1955</date>)</bibl>
    Source: <italic>Celebration at Vanderburgh</italic> (1987)
    TEI-XML: <bibl><title type="work">Celebration at Vanderburgh</title> (<date>1987</date>)</bibl>
    Source: <italic>Die Weisse Rose</italic>, a cantata for boys&#x0027; chorus, choir, narrators, baritone, and orchestra (1974&#x2013;75)
    TEI-XML: <bibl><title type="work">Die Weisse Rose</title>, a cantata for <instrumentation>boys' chorus, choir, narrators, baritone, and orchestra</instrumentation> (<date>1974–75</date>)</bibl>
    Within those <bibl> elements, if the only contents of parentheses is a four-digit date (or range) starting with "17", "18", "19", or "20", tag it as <date>. If the italicized title is followed by the keyword "for" tag the phrases following that keyword as <instrumentation>. The contents of that tag will be in one the following three patterns:
    thing
    thing and new thing
    thing, new thing, and newer thing
  4. Images should be contained in <figure> elements. Within those, captions (in Roman text) should be tagged in <p> elements, and acknowledgements (in italics, starting with "Courtesy" or similar) should be tagged in <byline> elements.
  5. Apostrophes in the source XML file are all represented in the source file as U+0027;. Please convert these to U+2019. *Please note that right single-quotes are already encoded as U+2019—you should convert these to <q rend="single"> before you convert the apostrophes so it can all be done automatically.
