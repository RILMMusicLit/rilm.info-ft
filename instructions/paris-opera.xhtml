The Paris Opéra
  1. General comments:
    • This title should be converted from scanned print.
    • The two volumes cover different time periods, so each has entries A-Z.
    • The two volumes have different subtitles and different copyright dates. See the example files for the distinction.
    • As with IEW, the bibliographies in each entry use abbreviations keyed to a list at the beginning of the volume.
  2. Front matter for volume 1 includes a title page, copyright page, dedication page, Preface, a very long Introduction, and a Abbreviations list.
    • Do not create a file for the Abbreviations. The contents will be incorporated into the individual entry files.
  3. Front matter for volume 2 includes a title page, copyright page, Preface, and Abbreviation list.
    • Again, do not create a file for the Abbreviations.
  4. Entries: include Works, Persons, Institutions, and Topics
    • A heading in bold italic type = <text type="work">:
      [Le Carnaval is a comedy...]
    • A heading in Bold Roman type containing a comma and followed by parentheses with "b." and "d." = <text type= "person">:
      [Charpentier, Marc (b. 1634; d. 24 February 1704, Paris), composer, left...]
    • Tag any entry heading that is not in italic type and has no parentheses following the bold heading as <text type="topic">.
    • Occupation(s) are given immediately after the closing parenthesis. Tag up to and including the occupation as a <rilmPerson>
      <p><rilmPerson><persName><surname>Charpentier</surname>, <forename>Marc</forename></persName> (<birth>b. <date when="1634" when-custom="1634">1634</date></birth>; <death>d. <date when="1704-02-24" when-custom="1704">24 February 1704</date>, <placeName>Paris</placeName></death>), <occupation>composer</occupation></rilmPerson>, left...</p>
  5. Bibliographies are headed "BIBLIOGRAPHY:" in small caps, and consist of abbreviations keyed to the list at the beginning of each volume, as well as written-out items. Most are followed by a page number or numbers.
    • Please replace the abbreviations in the entries with a <bibl> element containing the full marked-up bibliographic item from the list of abbreviations.
    • The items in the Abbreviations list follow this pattern: [Forename Surname, Title, edition/editor (Pubplace: Publisher, date), vols.] Mark names preceded by "ed." as <editor>. Note: some items may lack author, edition information, or volume numbers.
    :
    [BIBLIOGRAPHY: Lajarte I, 65; RGO V, 233-90.]
    <div type="ography">
    <head>BIBLIOGRAPHY:</head>
    <list>
    <item><bibl><author><forename>Theodore</forename> <nameLink>de</nameLink> <surname>Lajarte</surname></author>, <title>Bibliotheque musicale du theatre de l'Opera</title>, 2 vols.     </list>
    </div>
  6. Back matter for volume 1 includes 5 Appendices, Index, and About the Author.
    • APPENDICES:.
      • 1. The Repertory: Tag as introductory <p> followed by a <list> of abbreviations and a <list> of repertory. Tag the italic text as <title>. Open new <div>s for sections with headings such as "Lyric" and "Choreographic".
      • 2. Female Singers: Tag as introductory <p> followed by a <list>. Tag each <item> as a <rilmPerson>. Use the last name before the year as <surname>. If there is only one name given, tag only <persName>. Tag "Mme" as <roleName>; tag "aînée" and "cadette" as <genName>.
      • 3. Male Singers: Tag as introductory <p> followed by a <list>. Tag each <item> as a <rilmPerson>. Use the last name before the year as <surname>. If there is only one name given, tag only <persName>. Tag "aîné", "cadet", "fils", and "père" as <genName>.
      • 4. Female Dancers: as no. 2
      • 5. Male Dancers: as no. 3
    • INDEX: items in italics refer to entry titles, so they may be omitted. Process items in roman type per the general instructions for index terms.
  7. Back matter for volume 2 includes an Appendix, Index, and About the Author.
    • APPENDIX: The Repertory: tag as for vol. 1.
    • INDEX: items in italics refer to entry titles, so they may be omitted. Process items in roman type per the general instructions for index terms.
