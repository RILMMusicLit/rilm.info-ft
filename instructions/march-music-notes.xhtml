March Music Notes
  1. This title is to be converted from scanned hard copy.
  2. Front matter includes a title page; copyright page; dedication; Content Outline; Introduction; and list of Abbreviations.
    • Mark up the Abbreviations list as a two-column table.
  3. Back matter includes several Appendices and other sections:
    1. Lists of Contributors.
      • Set the individuals' list as a two-column <table>; Place the names (before the "--") in the left column and tag as <rilmPerson>; tag the content of the right column (after the "--") as <placeName>.
      • Make a new <div> at each geographic heading.
    2. March Popularity. Make a series of <lists>
    3. Key to Publishers. Set as a <list> and mark the parts as follows:
      [AB Allentown Band
      P.O. Box 1142
      Allentown, PA 18105 U.S.A.]
      <item><abbr>AB</abbr> <orgName>Allentown Band</orgName>, <address><street>P.O. Box 1142</street> <placeName><settlement>Allentown</settlement>, <region>PA</region></placeName> <postCode>18105</postCode></address></item>
    4. Key to Record Companies. Tag the same as Key to Publishers.
    5. Selected Bibliography. Note that first author's names are [surname, forename] and subsequent ones are [forename surname].
      • For the Periodicals section, tag only the <title> (underlined in print).
      • For the Unpublished Materials section, tag <author> and <title>
      • Set the Other section as a simple <list> with <item> children. Do not tag <bibl>.
    6. Arranger Index, Band Index and General Index. Insert <keywords> into the individual TEI entry files as per the general instructions.
    7. Title Index. Do not create a TEI file. The content will be found in the individual entries.
    8. Vita. Tag as a series of <p> elements.
  4. Entries include a heading, subheading, body and works list.
    • Use @rend="noMarker" for the <birth> and <death> elements and do not omit any text.
    • Tag bold face type in the body text as <title>
    • Tag the work lists as <div type="works"> and use <list> and <item> as usual.
      • Include only the underlined title and what is in parentheses in the <bibl>.
      • Tag what follows "Pub=" as <publisher><abbr>; tag the year or years as <date>.
      • Tag what follows "Rec=" as <idno type="label">.
      • Tag what follows the recording number as <orgName type="performing">
      • Enclose the descriptive paragraph in a <note> element. If there is an author name or names in parentheses at the end of the note, tag each as <persName>.

Please refer to the sample files.
