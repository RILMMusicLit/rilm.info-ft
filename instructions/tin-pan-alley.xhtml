Tin Pan Alley: An Encyclopedia of the Golden Age of American Song
  1. General comments:
    • This publication should be converted from the provided XML, artwork, and PDF.
    • Except for the front matter, each <sec> element in the source will correspond to one TEI file.
    • All <xref ref-type="page"> elements should be converted to the TEI element <pb>:
      <xref ref-type="page" id="page_453" />

      <pb n="453"/>
    • Convert all <italic> tags to <title>.
    • Ignore all <alt-title> elements and their contents.
    • American-type double-quotation marks are used.
      • Convert &#x201C; and &#x201D; to <q> and </q> tags;
      • Use American rule for single quotation marks.
    • Besides quotation marks, convert all &# codes to UTF-8 characters. No &# encodings should remain in the TEI files.
  2. Front matter will include the title page, copyright page, dedication, frontispiece, and Introduction.
    • The title page will correspond with the source elements <book-title-group> and <contrib-group>.
    • The copyright page will correspond with the source <publisher>, <isbn>, <pub-date>, <permissions>, and <imprint-meta> elements.
    • The dedication page content will correspond with the <notes notes-type="dedication"> of the source.
    • The frontispiece corresponds with <sec id="S1"> of the source.
    • The Introduction is tagged <sec id="S2" sec-type="fm-chapter"> in the source. Use the first <title> as the <div type="heading"> and subsequent <title> elements as <head> for a new <div type="section">.

    • All entries will have <div type="heading">, <div type="body">. A few entries will have <div type="works">.
    • The content of the first <title> element in the source will become the content of the <div type="heading">. For the <div type="heading">, tag the name as <rilmPerson> and <persName>; or <term> ;or <orgName>. The part of the personal name after the last space can be tagged <surname> and the others preceding it as <forename>:
      <div type="heading"><rilmPerson xml:id="r1"><persName><forename>Franck</forename> <forename><abbr>E.</abbr></forename> <surname>Churchill</surname></persName></rilmPerson></div>
    • The <p> element immediately following the article title is the start of the <div type="body">. For person entries, tag the first sentence as a <rilmPerson> element with an @corresp attribute pointing to the xml:id of the <rilmPerson> element above it. It begins with occupation(s), and continues with birth and death infomation in parentheses. Omit the parentheses and "b." and "d."
      <p>Composer (b. New York City, January 27, 1885; d. New York City, November 11, 1945). ... </p>
      <div type="body">
      <p><rilmPerson xml:id="r2" corresp="#r1"> <occupation>Composer</occupation> <birth><placeName>New York City</placeName><date when="1885-01-27" when-custom="1885">January 27, 1885</date></birth>; <death> <placeName>New York City</placeName><date when="1945-11-11" when-custom="1945">November 11, 1945</date></death></rilmPerson>. ... </p>
      </div>
  4. Cross-references are indicated only by <sc> tags in the source XML. Set the contents of <sc> and the preceding letter in a <ref> element, targeting the approriate article. Convert the contents of the <sc> elements to lower case and omit the <sc> tags.
    ...the G<sc>EORGE</sc> M. C<sc>OHAN</sc> musicals...
    ...the <ref target="#tpa0000?">George M. Cohan</ref> musicals...
  5. Back matter includes a Bibliography and two Indexes.

    • Treat the Indexes as usual back-of-book indexes.
