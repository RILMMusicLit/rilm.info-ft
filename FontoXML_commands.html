Clear markup Clean up the markup within the selection.
Collection title Contains the title of a book that is a collection of shorter works such as essays or stories.

Column(s) Contains the scope, in columns, of a bibliography reference.
Compiler In a bibliographic reference, contains the name(s) of a compiler, personal or corporate, of a work.

Contents section Contains a list of family members in a family article or a table of contents
Copy Copy is not supported as a button click. Use the keyboard shortcut instead.

Death Contains information about a person's death, such as its date and place.

Education Contains a description of the educational experience of a person.
Email address Contains an email adress.

Floruit Contains information about a person's period of activity.
Foreign text Identifies a word or phrase as belonging to some language other than that of the surrounding text.
Forename Contains a forename, given or baptismal name.
Gap Indicates a point where text is missing. Use if necessary symbols or characters cannot be found.
Generational name component Contains a name component used to distinguish otherwise similar names on the basis of the relative ages or generations of the persons named.
Generic title Contains the generic title of a work, such as 'symphony' or 'sonnet'
Genre Designates a genre in a works list.
Glossary list Insert a glossary list
Heading Contains a heading, usually the title of a section such as "Werke" or "Historische Untersuchungen"
Heading 1 Insert a new header/section pair at the first heading level.
Heading 2 Insert a new header/section pair at the second heading level.
Heading 3 Insert a new header/section pair at the third heading level.
Heading 4 Insert a new header/section pair at the fourth heading level.
Heading 5 Insert a new header/section pair at the fifth heading level.
Heading section Contains the article headwords and listing of contents
Highlighted text Marks a word or phrase as highlighted. Use only for the revision process.
Illustrator In a bibliographic reference, contains the name(s) of an illustrator, personal or corporate, of a work.
Image Indicates the location of an inline graphic, illustration, or figure.
Imprint Groups information relating to the publication or distribution of a bibliographic item.
Incipit Contains the incipit of a manuscript item, that is the opening words of the text proper, exclusive of any rubric which might precede it, of sufficient length to identify the work uniquely; such incipits were, in former times, frequently used a means of reference to a work, in place of a title.
Incipit title Contains a title taken from the first line of a work.
Indent Indent a list item or heading.
Indexed text Marks a location to be indexed for whatever purpose.
Instrumentation Contains instrumentation that appears with work titles.
Issue(s) Contains the issue number of a periodical in a bibliography reference.
Italic Contains italicized text.
Item in another document Refer to a specific piece of content in another document, for example a section or figure.
Item in this document Refer to a specific piece of content in this document, for example a section or figure.
Label Contains any label or heading used to identify part of a text, typically but not exclusively in a list or glossary.
Language known Summarizes the state of a person's linguistic competence, i.e., knowledge of a single language.
Line break Marks the start of a new (typographic) line in some edition or version of a text.
Line group Contains one or more verse lines functioning as a formal unit, e.g. a stanza, refrain, verse paragraph, etc.
Line(s) Contains the scope, in lines, of a bibliographic reference
Linguistic knowledge Summarizes the state of a person's linguistic knowledge, either as prose or by a list of elements.
List Contains an inline list. For use in works lists and bibliographies.
List of events Contains a list of descriptions, each of which provides information about an identifiable event.
List of organizations Contains a list of elements, each of which provides information about an identifiable organization.
List of places Contains a list of places, optionally followed by a list of relationships (other than containment) defined amongst them.
List of relations Provides information about relationships identified amongst people, places, and organizations, either informally as prose or as formally expressed relation links.
Media object Indicates the location of any form of external media such as an audio or video clip etc.
Meeting Contains the formalized descriptive title for a meeting or conference, for use in a bibliographic description for an item derived from such a meeting, or as a heading or preamble to publications emanating from it.
Mentioned Marks words or phrases mentioned, not used.
Monograph title Contains a title for a monograph.
Monographic level Contains bibliographic elements describing an item (e.g. a book or journal) published as an independent item (i.e. as a separate physical object).
Musical key Contains the name of a key of a musical work. E.g. A minor, Fis-moll.
Musical notation Insert a musical character that can not be easily inserted using your keyboard.
Musical note Contains the name of a musical note without reference to a particular octave or pitch.
Nationality Contains an informal description of a person's present or past nationality or citizenship.
Notated music example Contains a music example, usually in the form of an image.
Note Contains a note or annotation.
Occupation Contains an informal description of a person's trade, profession or occupation.
Ordered list Contains an enumerated list.

Organization name Contains an organization name.
Outdent Outdent a list item or heading.

Page(s) Contains the scope, in pages, of a bibliography reference.
Paragraph Marks paragraphs in prose.
Paragraph Remove the current heading, converting it to a paragraph.
Part(s) Contains the scope, in parts, of a bibliography reference.
Paste Paste is not supported as a button click. Use the keyboard shortcut instead.

Periodical title Contains the title of a journal, newspaper, or any other periodical.
Person Contains biographical information about a person. May contain a mix of text and elements.
Personal name Contains a proper noun or proper-noun phrase referring to a person, possibly including one or more of the person's forenames, surnames, honorifics, added names, etc.
Pitch Contains the name of a pitch in scientific pitch notation, Helmholtz pitch notation, or any other system.

Place name Contains an absolute or relative place name.

Relation Describes any kind of relationship or linkage amongst a specified group of places, events, persons, objects or other items.
Residence Describes a person's present or past places of residence.
Responsibility Contains a phrase describing the nature of a person's intellectual responsibility, or an organization's role in the production or distribution of a work.
