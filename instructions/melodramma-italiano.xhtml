Il melodramma italiano 1861-1900: Dizionario bio-bibliografico dei compositori (SSA)

Il melodramma italiano 1901-1925: Dizionario bio-bibliografico dei compositori (SSB)

  1. General comments:
    • This publication is in two parts: the first part (SSA) comprises one volume; the second part (SSB) comprises two print volumes contained in one PDF.
    • All entries are <text type="person">
    • Articles in the first part will have a <div type="heading"> and a <div type="ography>. Entries in SSB may also have a <div type="works">.
    • Italic type is used to indicate titles in body text and bibliographies, so tag with <title> rather than <hi>.
    • Where text is set in SMALL CAPS, convert to normal case.
    • Convert high quotation marks ( “ ” ) to <q> tags.
    • Guillemet-style quotes (« ») mark periodical titles. Tag the contents of « » as <title type="periodical">.
  2. SSA Front matter
    • Title page (OMIT the cover)
    • ISBN page: Do not create a TEI file for this page; append the ISBN itself to the Title-page file.
    • "Introduzione" (introduction)
    • "Ringraziamenti" (acknowledgements)
  3. ENTRIES begin with a numeral and heading in bold type followed by smaller Roman type in parentheses.
    • Personal names are formatted as Lastname [rolename] Forename [Forename], with no comma. Tag any uncapitalized word as <roleName> except "detto".
      • Create a TEI <title> by omitting the numeral, inserting a comma before the first space, and capitalizing the first letter of any all-lower case words (except "detto")
        Print: 39. Antolisei don Raffaele:
        <titleStmt><title>Antolisei, Don Raffaele</title></titleStmt>
        <div type="heading"><p>39. <rilmPerson><persName><surname>Antolisei</surname> <roleName>don</roleName> <forename>Raffaele</forename></persName></rilmPerson></p></div>
      • When the word "detto" appears, create the TEI <title> using only the part before "detto".
        Print: 49. Aspa Rosario detto Sari:
        <titleStmt><title>Aspa, Rosario</title></titleStmt>
        For the <div type="heading">, tag the part following the "detto" as <addName>. Don't tag the word "detto":
        <div type="heading"><p>49. <rilmPerson><persName><surname>Aspa</surname> <forename>Rosario</forename> detto <addName>Sari</addName></persName></rilmPerson></p></div>
    • Tag the birth and death information in parentheses as <birth> and <death> within the <rilmPerson> element (see examples).
    • Start <div type="body"> for the paragraph(s) of prose. Tag words in italic type as <title>, and the information in parentheses immediately following the title as <rilmPerformance>. Tag <placeName>, <orgName>, and <date> as shown in the example files. (Note: some place and theater names may fall outside the parentheses. If these can be caught, tag them, but it is acceptible if they are not.)
    • WORKLISTS (SSB only) are distinguished from the body text by smaller type and an italic word or phrase at the beginning.
      • Start a <div type="works"> and <list> here.
      • Tag each indented paragraph as <item><bibl>. Tag the italicized text as <title type="work">.
      • Where possible, tag personal names (capitalized) after "libretto di" as <author role="librettist">.
    • BIBLIOGRAPHIES in SSA are printed in one continuous paragraph. End the <div type="body"> and start a <div type="ography">.
      • Tag the word "Bibliografia" as <head>.
      • For the following <list>, start a new <item><bibl> at each semicolon (;).
      • Items in small caps that have no other information are abbreviations and do not need to be tagged further. Leave in all caps:
        [SCHMIDL;]
        <item><bibl>SCHMIDL</bibl></item>
      • For other items:
        • Tag text in small caps as <author>. Names are in forename-surname order. Normalize case.
        • Tag text in italic as <title>.
        • If there are two italic titles separated by "in" (not italic), tag the first <title type="article"> and the second as <title type="collection">.
        • Tag text in « » as <title type="periodical">.
        • Tag the text BEFORE the date as <publisher>; tag the text before the comma as <pubPlace>. See examples.
        • Tag personal names after "a cura" as <editor>.
        [Dizionario biografico dei liguri. Dalle origini al 1990, a cura di William Piastra, Genova, Consulta Ligure, 1994, II, pp. 183-184;]
        <item><bibl><title>Dizionario biografico dei liguri. Dalle origini al 1990</title>, a cura di <editor><persName><forename>William</forename> <surname>Piastra</surname></persName></editor>, <pubPlace>Genova</pubPlace>, <publisher>Consulta Ligure</publisher>, <biblScope unit="yr">1994</biblScope>, <biblScope unit="vol">II</biblScope>, pp. <biblScope unit="pp">183-184</biblScope></bibl></item>
        [DANIELE RUBBOLI, Basta un titolo per la gloria: "Il Bacio" di Arditi, "Visione veneziana" di Brogi, "Serenata-Rimpianto" di Toselli e "Lolita" di Buzzi-Peccia in La romanza italiana da salotto, a cura di Francesco Sanvitale, Torino, EDT - Istituto Nazionale Tostiano, 2002, pp. 468-474;]
        <item><bibl><author><persName><forename>Daniele</forename> <surname>Rubboli</surname></persName></author>, <title type="article">Basta un titolo per la gloria: <q>Il Bacio</q> di Arditi, <q>Visione veneziana</q> di Brogi, <q>Serenata-Rimpianto</q> di Toselli e <q>Lolita</q> di Buzzi-Peccia</title> in <title type="collection">La romanza italiana da salotto</title>, a cura di <editor><persName><forename>Francesco</forename> <surname>Sanvitale</surname></persName></editor>, <pubPlace>Torino</pubPlace>, <publisher>EDT - Istituto Nazionale Tostiano</publisher>, <biblScope unit="yr">2002</biblScope>, pp. <biblScope unit="pp">468-474</biblScope></bibl></item>
    • BIBLIOGRAPHIES in SSB are different from those in SSA: they are divided according to type of document or publication.
      • As above, use "Bibliografia" as the content of <head>.
      • Make a first <list> with the abbreviated entries.
      • After the first list, tag "Biografie ed epistolari." as <p>, then start a second <list> element.
      • Tag the next indented text up to the period (for example, "Studi e recenzione.") as <p>, begin the next <list>, and so on. See the example files.
      • Authors, titles, etc. are typeset just as in Part 1, and should be tagged the same as in Part 1.
