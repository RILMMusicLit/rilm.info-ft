Das Blasmusik-Lexikon (dbl)

  1. Most entries will be <text type="person">. Those can be identified by a heading with boldface surname followed by a non-boldface forename. Entries with headings that are entirely boldface should have <text type="topic">.
  2. Each entry contains a heading, a biographical paragraph, and (usually) one or more of the following lists: Werke (works), Bearbeitungen (arrangements), Märsche (marches), Schriften (writings), Aufsätze (essays), and Literatur (bibliography). Lists with the heading "Literatur" should be in a <div type="ography">. All others should be in a <div type="works">
  3. Bibliographies: Lists with the heading "Literatur" are bibliographies. These lists are set inline with each item separated by a semi-colon. Do not tag anythin in these lists; each item should go in a <bibl>.
  4. Works lists: All other lists (such as those with headings "Werke", "Bearbeitungen", "Märsche", "Schriften", or "Aufsätze") are works lists. These lists have one item per line. For these lists, tag all text up to the first comma as <title>. Tag a four-digit year at the end of the item as <date>; and tag anything between "bearb." and the next comma as <editor type="arranger">, as in the following examples:
    <item><bibl><title>Les Goyescas</title>, Intermezzo, bearb. <editor type="arranger">Pierre Dupont</editor>, Martin <date>1986</date></bibl>.</item>
    <item><bibl><title>Ouvertüre zu <q>Lucrezia Borgia</q></title>, bearb. <editor type="arranger">Dörle</editor>, Schulz (A = B).
  5. Front- and back matter: The following sections should each be set in a single file: "Zum Geleit" (p. 5), "Vorwort" (pp. 7-10), and "Abgekürzt zitierte Literatur" (pp. 13-18) following the instructions above.
  6. See references: If the name that is the source of a see-ref also appears in the heading of the destination entry, the source entry can be ignored. For example, If the source entry is
    Abel Lex, s. Max Leemann
    and heading of the destination entry is
    Leeman Max (Pseud. Lex Abel)
    The entry with "Abel Lex, s. Max Leemann" can simply be ignored.
