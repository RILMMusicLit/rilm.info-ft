Československý hudební slovník osob a institucí

This is a two-volume encyclopedia containing brief articles on persons and institutions in the former Czechoslovakia.

  1. General comments:
    • This publication should be converted from scanned print
    • High-low German style quotation marks („ ‟) are used.
    • In body text and worklists (but NOT in see-references), italic type is used to indicate titles, so tag with <title> rather than <hi>.
    • Note that the two volumes have different publication dates, namely 1963 and 1965. Note this for the TEI headers.
    • The two volumes also have slightly different front matter, so please encode and process both.
  2. Front matter
    • Half title page – OMIT
    • Dedication page
    • Title page
    • Editors / copyright page: Tag the three editors ("Redaktořy") as names
    • "Předmluva" (preface): Tag the last line "Brno...1962" as <byline>; author names as <docAuthor>.
    • "Poznámka redakce Státního hudebního vydavatelství"
    • "Slovníky a literatura" (bibliography), set as in a <div type="ography">, and create a list with a <bibl> in each <item>. Tag following this pattern:
      [D l a b a č  Bohumír: Allgemeines historisches Künstlerlexikon für Böhmen und zum Theil auch Mähren und Schlesien (Praha 1815, G. Haase, 3 svazky].
      <item><bibl><author><surname>Dlabač</surname> <forename>Bohumír</forename></author>: <title>Allgemeines historisches Künstlerlexikon für Böhmen und zum Theil auch Mähren und Schlesien</title> (<pubPlace>Praha</pubPlace> <date>1815</date>, <publisher>G. Haase</publisher>, <biblScope unit="vol" n="3">3 svazky</biblScope>.</bibl></item>
      (Close up the letterspacing of the author surname.)
    • "Šifry přispěvatelů" (contributors): After the initial paragraph, tag the list as a two-column <table>. Tag the content of the second <cell> as <persName> in forename(s)-surname order.
    • "Zkratky" (abbreviations): tag as a two-column table.
  3. Entries on persons can be distinguished from non-person articles by letterspaced text immediately following the bold text at the beginning of the article.
    • The surname is given first, in bold.
    • There is no comma between the surname and forename(s).
    • The (first) forename is set in letterspaced text. Subsequent forenames are set in normal, Roman text, followed by a comma.
      • Use the contents of the <persName> as the <title> in the TEI header.
      • Where there is more than one person by the same name, the publication inserts a superscript number and right parenthesis mark after the name. Add a left paren, drop the number to the baseline, move it before the comma, and add a space after the name. [ Červenka V á c l a v,1) český sborm., … ]
        <p><rilmPerson><persName><surname>Červenka</surname> <forename>Václav</forename></persName> (1), <nationality><abbr>čes.</abbr></nationality> <occupation>vojenský kap.</occupation>,
    • A few entries include an academic degree following the forename(s): "PhDr.", "JUDr." Tag as <roleName type="honorific"> if distinguishable.
    • If the text immediately following the comma is one of the following, tag as <nationality> "český", "česká", "čes.", "slovenský", "slovenská", "slov.". If is it "čech", "moravský", "moravská", "mor.", do not tag.
    • Tag the text after the name or nationality (if present), up to the next comma as <occupation>. Exception: if the name or nationality is followed by "čech", "moravský", "moravská", "mor.", begin the <occupation> element after it.
    • "n." is followed by birthdate - comma - birthplace - comma or period
    • "z." is followed by deathdate - comma - deathplace - period
      • Dates are in DD. MM(roman). YYYY format.
    Wrap all of the above in a <rilmPerson> element.

    [Czihalová-Funková I v e t a, slovenská operná speváčka (mezzosoprán), n. 14. IX. 1939, Žilina. … ]

    • In the print, there is no typographic break between the body text and any worklist or bibliography. Nevertheless, start a new <div type="works"> if you can identify the beginning of a worklist. There are two ways to do so:
      • Look for keywords "Skladby" or "Skladeb" or "Dila" and back up to the preceding period.
      • Look for letterspaced text, which is used for instrumentation.
      It is okay not to tag a worklist if there is any uncertainty about where it starts.
    • Start a <div type="ography"> if you can identify from keywords"Studie", "Práce", "Literarni", or "Knihy". These keywords themselves are often letterspaced.
  4. Personal name see references are in the form [Surname F o r e n a m e, v. Targetsurname Targetfore.] Treat these per http://rilm.info/ft/tei_rilm_instructions.html#see-refs.
  5. Institution entries do NOT have letterspaced text following the bold.
  6. Article Authors are sometimes indicated at the end of the article by a dash and initials in italic, followed by a right bracket.
    • Add a <div type="authors"> and a <docAuthor> element and repeat the initials in it. (see the examples).
  7. Each entry is followed by a symbol indicating the editor of the article:
    • ☉ = Gracian Černušák
    • ✻ = Bohumír Štědroň
    • ∆ = Zdenko Nováček
    Include the correct name in the <div type="authors"> as <docAuthor>. Precede the <persName> element by "Red. " (see the example files).
  8. There are Plates of photos between pp. 552 and 553 in vol. 1, and between pp. 672 and 673 in vol. 2. INSERT these into the articles to which they relate between the <div type="body"> and the following <div> (whether works, ography, or authors). The relationship should be clear according to the captions--where it is not, please query RILM contacts.
  9. Back matter consists of one page in each volume containing editorial and library cataloguing information.
