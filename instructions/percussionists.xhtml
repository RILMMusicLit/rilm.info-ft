Percussionists: A Biographical Dictionary (pbd)

  1. All entries should have <text type="person">.
  2. Each entry begins with a name (surname, forename(s), "addName") in allcaps, followed by birth and death information in parentheses. These two things together should be set in a <rilmPerson> element.
  3. In the birth and death information, "b." and "d." should be eliminated--the <birth> and <death> elements will take their place.
  4. Within the body text <div type="bio">, an asterisk denotes a cross reference.
  5. For any "ography" with the heading "Discography" or "Selected Discography", each line starts with a publisher name, followe by a identifying code. The publisher's name will either be spelled out in words (e.g. "Blue Note") or an initialism (e.g. "DMP"). There will often be another abbreviation after this name (e.g. "ACRE"), which is part of the indentifying code. That code consists of numbers, sometimes preceded by letters. Following the code, there is an italicized title, which should always be tagged <title type="work"> in Discographies. Following the title, there is often Roman text in parentheses. This is usually an author, and should be tagged <author> without the parentheses. Some examples [from p. 263 and 304]:
    • <bibl><publisher>Accurate</publisher><idno type="label">ACRE 5016</idno>: <title type="work">Falling from Grace</title> (<author>Plunge</author>, <date>1996</date>.</bibl>
    • <bibl><publisher>ECM</publisher><idno type="label">1073</idno>: <title type="work">Bright Size Life</title> (<author>Pat Metheny</author>), <date>1975</date>.</bibl>
    • <bibl><publisher>PM</publisher><idno type="label">PMR001</idno>: <title type="work">Open Sky</title>. <date>1973</date>.</bibl>
    • <bibl><publisher>RMD</publisher>/<publisher>RMC</publisher><idno type="label">81208</idno>: <title type="work">In Session</title> (<author>Tito Puente’s Golden Latin Jazz All-Stars</author>), <date>1994</date>.</bibl>
    If the Roman text in parentheses starts with a lowercase letter, start the author element with the first capitalized word, as in the following example (from p. 302):
    • <bibl><publisher>University of Illinois</publisher><idno type="label">CR53</idno>: <title type="work">School of Music</title> (cond. <author>University of Illinois Perc. Ens.</author>), <date>1975</date>.</bibl>
    If the Roman text in parentheses is a date, please use <date> instead of <title>
  6. For any "ography" with the heading of "Videography" or "Selected Videography", follow the same instructions as for "Discography", except that material within parenthese should not be tagged. Also, the following caveat: there are often multiple publishers for videos, and often no identifying code. A couple examples, from p. 273 and 284:
    • <bibl><publisher>Kultur Video</publisher>: <title type="work">Red Norvo</title> (with Tal Farlow), <date>1982</date>.</bibl>
    • <bibl><publisher>Audio Visual Images</publisher>/<publisher>New Dimension Media</publisher>: <title type="work">The Dreamer That Remains: A Portrait of Harry Partch</title>, <date>1974</date> (produced by Betty Freeman and directed by Stephen Pouliot)</bibl>
