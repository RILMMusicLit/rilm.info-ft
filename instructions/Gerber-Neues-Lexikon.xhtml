Neues historisch-biographisches Lexikon der Tonkünstler (1812-1814)
  1. General comments:
    • This publication will be converted from supplied PDF.
    • Text is mainly in gothic script, specifically Walbaum Fraktur type, but non-German words are printed in Roman type. Some portions of the text are illegible in the PDF; these passages should be represented with an empty <gap> element.
    • Every page has a catchword, and some have additional printer's marks, as the bottom line; OMIT these.
    • Personal names are distinguished by loose letter spacing.
    • There are four volumes, dated 1812, 1812, 1813, and 1814. The end of the fourth volume is an appendix.
    • All entries are <text type="person">.
    • Entries will NOT have <div type="heading"> or <div type="subheading">. All will have <div type="body"> and some will have one or more <div type="works">
    • Low-high German-style quotation marks („ ‟) are used. See below for special treatment of block quotes.
  2. Front matter will include a title page, a "Pränumeranten-Verzeichniß", and lengthy "Vorrede" from volume 1; and the title page and "Pränumeranten-Verzeichniß" for the subsequent three volumes. Skip half-title pages.
  3. Personal names as the titles of entries are given in surname-forename order. The surname is in letter-spaced type; forename(s), if any, is (are) given in parentheses.
    • (...) means that the forename is unknown. If there is no forename at all—usually the case with ancient names—there are no parentheses, and the name should be tagged <persName> only, without <forename> or <surname>. Examples: Antippus, Antisthenes.
    • IF there is an alternate name, it is separated by "oder" ("or"): [Allatius oder Allacci (Leo)]
    • Omit parentheses ( ) from the <title> element in the TEI header, and add a comma following the surname, but KEEP the parentheses in the <div type="body"> and do not add a comma.
  4. Long quotes are indicated in the print by a low double-quotation mark repeated at the beginning of each line of the quotation:
    • [...ihm schreibt: „Es war auch vor einigen
      „Jahren in Engländischer Sänger, Na-
      „mens Abell, in vieler Hochachtun, und
      „ließ sich sowohl in Holland...
      „und Trinken sehr viel half.‟ — Bey diesem... ]
    Tag the quotation as <cit rend="block">:
    <p>...ihm schreibt: </p>
    <cit rend="block"><quote>Es war auch vor einigen Jahren in Engländischer Sänger, Namens Abell, in vieler Hochachtun, und ließ sich sowohl in Holland... und Trinken sehr viel half.</quote></cit>
    <p>Bey diesem...</p>
  5. See references are in the form [Surname (Forename), s. Target.] Treat these according to the general instructions; add as keywords in the TEI header to the target file.
  6. Worklists are usually run inline with the preceding text, but are delimited by numerals with a following parenthesis mark. Make a <div type="works">, and tag each numbered work as <item> in a <list>. Long worklists may have dividing headers with Roman numerals. Please see example files.
  7. A large Appendix ("Anhang") contains several distinct indexes; each should be in a separate TEI file:
    1. Vorerinnerung (foreword): include the "Anhang" title page and "Inhalt des Anhanges"
    2. Verzeichniß in Kupfer gestochener und in Holz geschnittener Bildnisse berühmter Tonlehrer und Tonkünstler
      • Set the opening paragraph—including the word "Verzeichnis" but not the Roman number I—in <div type="body">
      • Make a <div type="section"> for each letter (A,B,C, etc.) and tag the letter as <heading>.
      • Within each letter, create a list element with an item for each name; tag the name as such.
        <item>*<persName><surname>Aaron</surname> (<forename>Pietr.</forename>)</persName> s. a. Lex. Anh. 2 mal.</item>
    3. Fortgesetztes Verzeichniß einiger Gemälde nnd Zeichnungen von Bildnissen berühmter Tonlehrer und Tonkünstler.
      • Set the opening paragraph, starting with the word "Fortgesetztes", in <div type="body">
      • Tag the remainder in <div type="section"> as a <list> with an <item> for each name; tag the name.
    4. Fortgesetztes Verzeichniß von Statüen und Büsten großer Tonlehrer und Tonkünstler, auch von Abdrücken und Medaillen, mit Bildnissen derselben.
      • Tag same as the preceding.
    5. Fortgesetztes Verzeichniß von großen und berühmten Orgeln, deren Risse durch den Grabstichel gemein gemacht worben sind.
      • Tag the opening paragraph as above.
      • Tag the remainder in <div type="section"> as a <list> with an <item> for each entry; tag the first word of each entry as <placeName>.
    6. Register über Erfindungen zur Vervollkommung der Kunst und der Instrumente. This is an index of terms pointing to name entries in the main dictionary. It also contains see-references to itself.
    7. Berichtigungen und Zusätze zum ganzen Werke. These are corrections and additions to main entries.
      • Create a separate TEI file for each; append "[supplement]" to the contents of <title> in the <TEIheader>.

Please refer to example files.
