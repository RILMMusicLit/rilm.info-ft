International Encyclopedia of Women Composers
  1. General comments:
    • This title should be converted from scanned print.
    • Entries include a heading, a short biography, and a works list in sections. Much more information is given in Appendices and other Back matter, including Bibliography, Discographies, photographs, and alternate names.. We would like to pull this information together into the TEI file for each main entry.. Specifics are given below.
  2. Front matter for Vol. 1 comprises a frontispiece; title page; copyright page; dedication page; Foreword; Preface - 2nd Edition; Preface - 1st Edition; Acknowledgments; Notable facts...; Quotations; Statistics; Guide to Use; and Abbreviations.
    • Tag the Abbreviations list as a <table> with two columns
    • Note: regard the "Addendum" section as part of the main body of the encylopedia.
  3. Entry headings give the personal name in surname, forename order; The surname is set in bold, all capitals; the forename is set in bold, normal case.
    • Tag name forms in parentheses in separate <persName> elements, but the same <rilmPerson>.
      [HOWE, Mary Alberta Bruce (Mary Carlisle Howe) (Mrs. Walter Bruce)]
      <div type="heading"><p><rilmPerson><persName><surname>HOWE</surname>, <forename>Mary</forename> <forename>Alberta</forename> <forename>Bruce</forename></persName> (<persName>Mary Carlisle Howe</persName>) (<persName>Mrs. Walter Bruce</persName>)</rilmPerson></p></div>
    • The biographical paragraph that consitutes the <div type="body"> begins one of two ways:
      1. If a specific birth date is available, the paragraph begins "Nationality occupation(s). b. Birthplace, birthdate."
      2. If NO specific birth date is available, it begins "Century Nationality occupation(s)."
      [KATS-CHERNIN, Elena
      Australian pianist, teacher and composer. b. Tashkent, USSR, November 4, 1957.]
      <div type="heading"><p><rilmPerson xml:id="r1"><persName><surname>KATS-CHERNIN</surname>, <forename>Elena</forename></persName></rilmPerson></p></div>
      <div type="body"><p><rilmPerson xml:id="r2" corresp="#r1"><nationality>Australian</nationality> <occupation>pianist</occupation>, <occupation>teacher</occupation> and <occupation>composer</occupation>. <birth>b. <placeName>Tashkent, USSR</placeName>, <date when="1957-11-04" when-custom="1957">November 4, 1957</date></birth></rilmPerson>....</p></div>
      [KATTS, Letty
      20th-century Australian composer.]
      <div type="heading"><p><rilmPerson xml:id="r1"><persName><surname>KATTS</surname>, <forename>Letty</forename></persName></rilmPerson></p></div>
      <div type="body"><p><rilmPerson xml:id="r2" corresp="#r1">20th-century <nationality>Australian</nationality> <occupation>composer</occupation></rilmPerson></p></div>.
  4. Discographies: Entries whose body paragraph has "DISCOGRAPHY" in the last line are represented in a "Recordings" list in vol. 2, beginning on p. 1069. Entries in the Recordings list are sorted by name and have the same headings as the main encyclopedia entries, but with dates added in parentheses. Recordings under each name are listed by work title in all caps.
    • Mark up the recordings entries in vol. 2 according to this pattern:
      [TITLE
      Performer(s) (voice or instrument) LABEL IDNO].
      Note: not all entries contain all kinds of data.
    • Insert the entries in a <div type="ography"> in the main encylopedia entry.
    Refer also to the example files.
  5. Figures: Entries whose body paragraph has "PHOTOGRAPH" in the last line have a related figure (portrait) in the section of Vol. 2 beginning p. 817 (PDF page 217). The photo captions in print are the same as the headings of the main encylopedia entries.
    • Insert a <figure> element between <div> elements in the XML file that references the photograph. See the example files.

  7. Bibliographies are headed "Ref." in non-bold type, and consist of numerals keyed to the bibliography at the back of vol. 2 (beginning page 783).
    • Please replace the numbers in the entries with a <bibl> element containing the full marked-up bibliographic item from vol. 2, pages 783- 793.
    • The items follow this pattern: [Surname, Forename. Title. Pubplace: Publisher, date.]. Mark names followed by ", ed." as <editor>. Note: some entries lack place, publisher, or date.
    :
    [Ref. 171, 234]

  8. See references are in the form [SURNAME, Forename. See SURNAME2, Forename2]"
  9. Back matter is very extensive, but not all of it needs to be marked up.
    • BIBLIOGRAPHY:
      • Numerical References: Do not mark up; to be included with individual entry files.
      • Alphabetical by Author: Do not mark up; sorted copy of preceding
      • Alphabetical by Title: Do not mark up; sorted copy of preceding.
      • For Further Reading: DO mark up as a bibliography
    • PHOTOGRAPHS: included with individual entry files
    • APPENDICES:.
      • 1. Information Wanted: Tag as introductory <p> followed by a <list> for each "century" given. Tag the contents of each <item> as a <rilmPerson>, <persName>, etc.
      • 2. Music Key Signatures in 25 Languages: Tag as two <table>s with 14 columns each.
      • 3. Comparative distribution of Composers by Century: Scan and tag as <figure>s
      • 4. Composers by Country...: Make a <div type="section"> for each country and a <list> for each century.
      • 5. List of Pseudonyms: Treat as an index; specifically, insert the text from the "Pseudonym" column in a <keywords> element in the TEI header of the entry for the name in the "Composer" column.
      • 6. Operas and Operettas by Women Composers: Do not mark up; included in main entries worklists.
      • 7. Composers Influenced by Shakespeare: Do not mark up. Add "Shakespeare" in <keywords> of the TEI header of the main entry for each name in bold type.
      • 8. Composers by Instrument and Music Form: Make a <div type="section"> for each item in all caps, and a <list> for each century.
      • 9. Composers by Calling, Occupation and Profession: Make a <div type="section"> for each item in all caps, and a <list> for each century.
      • 10. Composers by the Instruments they Play: Make a <div type="section"> for each item in all caps, and a <list> for each century.
      • 11. International Council of Women and Affiliates: Tag prose as <p>, remainder as a <list> with an <item> for each country/addess.
      • 12. International Music Societies and Information Centers: Tag prose as <p>; make a <list> for each country (in all capitals), and an <item> for each address.
      • 13. International Performing and Music Rights Societies: Same as Appendix 12.
      • 14. National Federation of Music Clubs: Tag prose as <p>; make a <list> for each State (in all capitals), and an <item> for each address.
    • DISCOGRAPHY: .
      • Preface: Mark up as prose.
      • Recordings: Do not mark up; to be included with main entry files.
      • Record Companies: Tag as a <table> with three columns.
